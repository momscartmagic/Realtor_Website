# Elevate Real Estate Website

A premium, modern, and fully responsive web application for real estate professionals to showcase their services, property listings, and brand.

## Features
- **Responsive Design:** Fully scaled layout optimized for mobile, tablet, and desktop viewports.
- **Dynamic Routing:** Smooth page transitions using `react-router-dom`.
- **Properties Showcase:** Elegantly designed grids to display luxury real estate listings.
- **Service Offerings:** Highlight key real estate services tailored to clients.
- **Contact Forms & Integration:** User-friendly contact sections to generate leads.
- **Modern UI Elements:** Glassmorphism, blurred navbars, and subtle fade-in CSS animations.

## Tech Stack
- React (18.2.x)
- React Router DOM (6.21.x)
- React Icons (4.12.x)
- Vite (5.0.x)
- Custom CSS Variables, Flexbox, & Grid Layouts

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/momscartmagic/Realtor_Website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd realtor-website
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

### Running the Development Server
To start the app locally:
```bash
npm run dev
```
The application will be accessible at `http://localhost:5173/` by default.

### Building for Production
To build the application for production deployment:
```bash
npm run build
```
This will generate optimized static assets in the `dist` directory. You can preview the built files by running `npm run preview`.

## Project Structure
- `src/components/` - Reusable UI components like Navbar, Footer, and Floating Contact widgets.
- `src/pages/` - Core application pages: Home, About, Services, Properties, and Contact.
- `src/index.css` - Global CSS variables, base styles, and utility classes.
